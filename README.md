GIFts
==================================
_requires Node.js >=6.0_

Getting Started
---------------
```sh
# clone repository
$ git clone git@github.com:preichelt/GIFts.git

# navigate to project
$ cd GIFts

# install dependencies
$ npm install
```

Lifecycle
---------------
```sh
# start app in development mode
$ npm start

# start app in production mode
$ npm run start:prod

# restart app
$ npm restart

# stop app
$ npm stop
```

Specs
---------------
```sh
# run unit tests
$ npm test
```

Diagnostics
---------------
```sh
# check app status
$ npm run status

# tail app logs
$ npm run logs

# flush app logs
$ npm run logs:flush

# update/fix pm2
$ npm run update
```
